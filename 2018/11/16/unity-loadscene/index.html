Unity3D 重载/加载 scene

发表于 2018-11-16

说明：本人使用的 unity 版本为 2017.2.0f3

很多时候我们会需要在游戏结束，重新开始的时候，从当前状态回到场景初始状态。下面一步步讲解。

  1. 首先使用 UnityEngine.SceneManagement 命名空间，方便调用

  2. 通过使用 LoadScene 方法加载一个新的场景。可传入 场景名称/路径 或 sceneBuildIndex。LoadScene 还有第二个参数 mode，选择加载模式，默认为 single。mode 有 Single 和 Additive 两种。Single 模式下，会关闭当前所有场景并加载需要的场景。Additive 模式下，会把需要加载的场景直接添加到当前场景中。

  3. 使用 LoadScene 的时候，场景的加载不会立即完成，会在下一帧的时候完成。这种非异步的方式可能会造成场景的混乱或者其他令人困惑的问题。所以建议使用异步的方式，使用 LoadSceneAsync 方法代替 LoadScene。使用方式与 LoadScene 一样，该方法会在后台异步加载场景。
